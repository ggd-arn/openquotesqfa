<h1>openquotes - quotes by someone for everyone</h1>
<p>openquotes is a website/opensource repo where you can load random motivational quotes. <strong>Useless, but useful. Usefully useless.</strong></p>
<img width="1386" height="876" alt="image" src="https://github.com/user-attachments/assets/e545d176-4777-4dac-821f-dfd793249d1e" />


<h2>Features include:</h2>
<ul>
  <li>Quote copying to clipboard</li>
  <li>Quotebook (unlock discovered quotes for completion</li>
  <li>The Textbox of Wisdom (get secret easter eggs!)</li>
  <li>Biographies (information for each quotee/background behind each quote)</li>
</ul>

<h2>Textbox Codes:</h2>
<ul>
  <li>"rls" - Resets local storage, removes all discovered quotes.</li>
  <li>"Oh, Neko!" - Spawns the oneko cat. Silly!</li>
  <li>"Vibe along and find out" - Loads a vibecoded/styled theme.</li>
  <li>"Retro" - Loads a Windows 95-styled UI.</li>
  <li>...the rest you'll have to find your own</li>
</ul>
